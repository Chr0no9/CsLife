## **Naming Conventions**

* Variables and functions should use camelCase.  
  * Ex.  
    * playerName  
    * currentQuarter  
    * updateStats()  
    * nextQuarter()  
* Classes should use PascalCase.  
  * Ex.  
    * Character  
    * Quarter  
    * Event  
* Constants should use UPPER\_SNAKE\_CASE.  
  * Ex.  
    * MAX\_HEALTH  
    * STARTING\_POINTS

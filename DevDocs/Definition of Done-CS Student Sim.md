## **Definition of Done for Tasks**

* The feature is fully implemented according to the user story.  
* The system functions correctly without major bugs.  
* The code has been reviewed and merged into the main branch.  
* The feature integrates and works correctly with the rest of the game.  
* User interface elements are clean and function correctly.  
* The project builds and runs without errors.  
* Basic testing has been completed by at least one team member.

## **Definition of Done for User Stories**

* The feature is fully implemented according to the user story.  
* The system functions correctly without major bugs.  
* All associated tasks have met the Definition of Done.  
* The feature has been integrated with the rest of the application.  
* The feature has been tested from the user's perspective and functions correctly.  
* Basic testing has been completed by at least one team member.  
* Relevant documentation and required documentation has been updated.
